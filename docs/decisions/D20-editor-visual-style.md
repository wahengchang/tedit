# 決議 D20:編輯器視覺風格 = 深色專業風(Figma 風)

> 日期:2026-06-14。使用者拍板(M4 動工前)。

## 裁決

| 項 | 結論 |
|----|------|
| 調性 | **深色專業風**(Figma 風);底色 #1e1e1e 系深灰、細邊框、緊湊工具感 |
| 佈局 | 工具列(頂)/ 圖層列表(左)/ 畫布(置中,深灰底襯托作品)/ 屬性面板(右) |
| 取捨 | 否決淺色 Canva 風與極簡灰階風 |

## 佈局線框

```
┌─────────────────────────────────────────────┐
│ tedit   文字 圖片 形狀                [存檔]   │ ← 深灰工具列
├──────┬───────────────────────────┬──────────┤
│ 圖層  │                           │ 屬性      │
│ ▦標題 │       ┌───────────┐       │ X[ ] Y[ ]│
│ ▦圖片 │       │   畫布      │      │ 寬[ ]高[ ]│
│ ▦背景 │       │ (深灰背景)  │      │ 字級[28] │
│      │       └───────────┘       │ ◉ 綁定變數│
└──────┴───────────────────────────┴──────────┘
```

## 給 M4 的指示

- 配色基線:面板/工具列 #1e1e1e~#252526,邊框 #3a3a3a,文字 #ddd,強調色一枚(選取/綁定角標)。
- 畫布區用比面板更深或中性的襯底,讓作品(常為亮色)跳出來。
- 圖層列表在左、屬性面板在右,選取畫布元素 → 兩側同步高亮/填表。
- 風格僅 UI 層,**不影響 engine 場景與同像素**(D11 不受牽動)。

## 連帶

此決議只定「編輯器外觀」。畫布內互動沿用 fabric 內建(D13/D18:選取框、控制柄、IText 編輯,免 workaround)。
